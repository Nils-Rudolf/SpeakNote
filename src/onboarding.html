Welcome to SpeakNote

1Allow Microphone Access

SpeakNote needs access to your microphone to record and transcribe speech.

You will be directed to System Settings > Privacy & Security > Microphone.

2Enable Accessibility Features

SpeakNote needs accessibility access to insert text into other applications.

You will be directed to System Settings > Privacy & Security > Accessibility.


3Set Up API Key

SpeakNote uses an API for speech recognition. Please enter your API key.

You can skip this step and add the API key later in settings.

4Ready to Go!

SpeakNote is now set up and ready to use!

Press F5 to start recording, and F5 again to stop and transcribe.

You can adjust settings anytime from the menu bar icon.
